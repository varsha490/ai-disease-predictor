About

This project is a simple, stable, multi-page web demo. It uses: Flask for the backend API and scikit-learn with a small, hardcoded training set using LogisticRegression.

The prediction is shown with a confidence score and a probability chart. A local browser history keeps your last 5 predictions.

Important: The results are estimates for demonstration purposes only. They are not medical advice.
